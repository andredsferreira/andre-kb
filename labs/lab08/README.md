### LAB08: Containerizing Go Application

**Description**



**Running**

This command runs the application and also rebuilds (remove the
--build flag if you don't want to rebuild, just remember that any
modifications to the compose.yaml will need to be rebuilt in order to
be seen).

NOTE: Run the command on this directory.

```bash
docker compose up --build -d
```

**Stopping**

```bash
docker compose down
```
